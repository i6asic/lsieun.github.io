## 安装

### 下载

```text
https://services.gradle.org/distributions/
```

### 解压

解压 `gradle-8.6-bin.zip` 到：

```text
C:\Gradle\gradle-8.6
```

### 环境变量

添加环境变量：

```text
C:\Gradle\gradle-8.6\bin
```
