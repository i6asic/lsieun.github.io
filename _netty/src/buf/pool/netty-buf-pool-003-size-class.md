## 不同的内存规格

### 分类

在 Netty 中，内存规格分为三类： Small、Normal 和 Huge；这三个内存类别是由 jemalloc 演化而来。

![](/assets/images/netty/buf/netty-buffer-pool-jemalloc-category.svg)

形象类比：小人族、普通人、巨人族。

![](/assets/images/cartoon/one-piece-001.png)

- [Races and Tribes](https://onepiece.fandom.com/wiki/Category:Races_and_Tribes)

    - [Humans](https://onepiece.fandom.com/wiki/Humans)
    - [Dwarves](https://onepiece.fandom.com/wiki/Dwarves)

### 分类量化

```text
| Category | Size             |
|----------|------------------|
| Small    | 0KB < x <= 28 KB |
| Normal   | 28 KB < x <= 4MB |
| Huge     | 4MB < x          |
```

![](/assets/images/netty/buf/netty-buffer-pool-size-class-capacity.svg)

### 标准化

```text
┌─────────────┬───────────────┐
│    User     │     Netty     │
├─────────────┼───────────────┤
│ reqCapacity │ normCapacity  │
└─────────────┴───────────────┘
```

```java
import io.netty.buffer.SizeClasses;
import lsieun.cst.MyConst;
import lsieun.drawing.theme.table.TableType;
import lsieun.drawing.utils.TableUtils;

public class HelloWorld {
    public static void main(String[] args) {
        // SizeClasses
        SizeClasses sizeClasses = new SizeClasses(
                MyConst.pageSize, MyConst.pageShifts,
                MyConst.chunkSize, MyConst.directMemoryCacheAlignment
        );

        // reqCapacity --> normalizedCapacity
        int reqCapacity = 0;
        int normCapacity = sizeClasses.normalizeSize(reqCapacity);

        // print
        String[][] matrix = {
                {"reqCapacity", "normCapacity"},
                {String.valueOf(reqCapacity), String.valueOf(normCapacity)}
        };
        TableUtils.printTable(matrix, TableType.MARKDOWN);
    }
}
```

| reqCapacity | normCapacity |
|-------------|--------------|
| 0           | 16           |

## SizeClass

```java
abstract class PoolArena<T> implements PoolArenaMetric {
    enum SizeClass {
        Small,
        Normal
    }
    
    private static SizeClass sizeClass(long handle) {
        return isSubpage(handle) ? SizeClass.Small : SizeClass.Normal;
    }
}
```

格式：

```text

```

计算大小：

```text
size = (1 << log2Group) + (1 << log2Delta) * nDelta
```
