---
title: "Nginx Command"
sequence: "103"
---

```text
$ whereis nginx
nginx: /usr/sbin/nginx /usr/lib/nginx /etc/nginx /usr/share/nginx
```

查看 Nginx 的版本号：

```text
nginx -v
```

启动 Nginx：

```text
nginx
```

关闭 Nginx：

```text
nginx -s stop
```

重新加载 Nginx：

```text
nginx -s reload
```

检查配置文件：

```text
nginx -t
```
