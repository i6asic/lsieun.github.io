## 易经符号

```java
public class YiJing {
    public static void main(String[] args) {
        // 两仪
        UnicodeUtils.print(0X268A, 0X268B);
        // 四象
        UnicodeUtils.print(0X268C, 0X268F);
        // 八卦
        UnicodeUtils.print(0X2630, 0X2637);
        // 易经六十四卦符号
        UnicodeUtils.print(0X4DC0, 0X4DFF);
    }
}
```
