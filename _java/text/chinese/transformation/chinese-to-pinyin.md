```text
pinyin4j
优点： 轻量级，易于集成和使用。
局限性： 基本的多音字支持，不能根据上下文自动判断正确读音。

hanLP
优点： 功能丰富，提供准确的多音字处理和其他自然语言处理能力。
局限性： 相对较重，需要更多资源。
```

## 引入依赖

```xml
<dependency>
    <groupId>com.belerweb</groupId>
    <artifactId>pinyin4j</artifactId>
    <version>2.5.1</version>
</dependency>
```
