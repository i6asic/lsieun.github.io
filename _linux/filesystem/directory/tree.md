## 安装

Ubuntu or Debian:

```text
$ sudo apt install tree
```

RedHat or CentOS:

```text
$ sudo yum -y install tree
```

## 查看帮助

```text
$ man tree
```

## 使用

```text
$ tree -L 1 /
/
├── bin -> usr/bin
├── boot
├── dev
├── etc
├── home
├── lib -> usr/lib
├── lib64 -> usr/lib64
├── media
├── mnt
├── opt
├── proc
├── root
├── run
├── sbin -> usr/sbin
├── srv
├── sys
├── tmp
├── usr
└── var
```
