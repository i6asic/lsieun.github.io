## 養身

```text
人知飲食足以益吾身，而不知節飲食乃所以養吾身。——《格言聯璧·攝生類》

省嗜欲，減思慮，戒忿怒，節飲食，此養身也。
規利害，避勞怨，營窟宅，守妻子，此惜身也。——《格言聯璧·攝生類》
```

## 卻病

```text
飲食知節，則疾病少。——《格言聯璧·攝生類》

慎風寒，節飲食，是從吾身上卻病法。
寡嗜欲，戒煩惱，是從吾心上卻病法。——《格言聯璧·攝生類》
```

## 節飲食

```text
性分不可使不足，故其取數也宜多：曰窮理，曰盡性，曰達天，曰入神，曰致廣大，極高明。
情欲不可使有餘，故其取數也宜少：曰謹言，曰慎行，曰約已，曰清心，曰節飲食，寡嗜欲。——《格言聯璧·存養類》

① 性分：天性，本性。

对人天性的培养不可以不充分，所以讲究的地方也应当很多，包括：
穷极天下万事之理，
究尽世间万物所禀之性，
发现天地运化的规律，
探查自然内在的神奇，
达到胸怀宽广、气象博大、目光高远、思维明睿的境界。
情感欲望不可以太多，所以应当减少些，包括：
说话谨慎，
行事小心，
约束自己，
清净内心，
节制饮食，
戒除不良嗜好。
```

## 戒飽食

```text
戒浩飲，浩飲傷神。戒貪色，貪色滅神。    <--
戒厚味，厚味昏神。戒飽食，飽食悶神。    <--
戒多動，多動亂神。戒多言，多言損神。
戒多憂，多憂郁神。戒多思，多思撓神。
戒久睡，久睡倦神。戒久讀，久讀苦神。——《格言聯璧·學問類》
```

## 休食

```text
難消之味休食，難得之物休蓄。
難酬之恩休受，難久之友休交。
難再之時休失，難守之財休積。
難雪之謗休辯，難釋之忿休較。——《格言聯璧·攝生類》
```
