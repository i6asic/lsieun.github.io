---
title: "Cell Style"
sequence: "101"
---


Take note that `CellStyle` has several formatting properties that we can modify:

```html

<table border="1">
    <tbody>
    <tr>
        <th>Property</th>
        <th>Description</th>
    </tr>
    <tr>
        <td><em>DataFormat</em></td>
        <td>Data format for the cell, such as date</td>
    </tr>
    <tr>
        <td><em>Alignment</em></td>
        <td>Type of horizontal alignment for the cell</td>
    </tr>
    <tr>
        <td><em>Hidden</em></td>
        <td>Whether the cell is to be hidden</td>
    </tr>
    <tr>
        <td><em><span>Indention</span></em></td>
        <td>Number of spaces to indent the text in the cell</td>
    </tr>
    <tr>
        <td><em>BorderBottom</em>,<p></p>
            <p><em>BorderLeft</em>,</p>
            <p><em>BorderRight</em>,</p>
            <p><em>BorderTop</em></p></td>
        <td>Type of border to use for the bottom, left, right, and top border of the cell</td>
    </tr>
    <tr>
        <td><em>Font</em></td>
        <td>Font property for this style, such as font color</td>
    </tr>
    </tbody>
</table>
```
