## 引入依赖

引入 `junit-jupiter-params` 依赖：

```xml
<junit.version>5.11.3</junit.version>
```

```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter-params</artifactId>
    <version>${junit.version}</version>
    <scope>test</scope>
</dependency>
```
